# LeadHunter Frontend

Premium SaaS frontend built with Next.js App Router and Tailwind CSS.

## Commands

- `npm run dev` - start development server
- `npm run build` - create production build
- `npm run start` - run production server
- `npm run lint` - run lint checks

## Structure

- `src/app` - routes and layouts
- `src/modules` - domain modules (auth, leads, campaigns, etc.)
- `src/components` - shared reusable UI/layout components
- `src/lib` - utilities and shared libraries
- `src/styles` - global styles and design tokens

## Current UI Scope

- `signin` page
- `signup` page

Frontend-only implementation (no backend logic).
